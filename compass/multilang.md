# Multi-language evidence: CodeHealth guidance beyond Python

<span style="font-size: 0.85em; opacity: 0.7;">
<a href="/compass/">Back to overall page</a>
</span>

This page provides supporting evidence that the value of using <a href="/compass/#codehealth">CodeHealth</a> as a compass for coding agents generalizes beyond Python—specifically to C++ and Java—in large-scale experiments using a self-hosted agent and medium-sized LLMs.
